# Система бронирования мест на контрольной и сдаче домашнего задания

Веб сервис для студентов, позволяющий бронировать места в аудиториях на контрольных работах, используя баллы, заработанные за академическую взаимопомощь.

## Технологии
- Python
- PostgreSQL
- Docker
- Pytest

### Цель

Грамотно реализовать данный проект используя паттерны, заботать хорошую архитектуру проекта
